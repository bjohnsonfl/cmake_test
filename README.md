# CMAKE Build

```bash
# clone the repo and cd to root dir

# create a build directory and cd into it
mkdir build
cd build

# run cmake elaboration 
cmake ..

# build the program with the generated make files
make

# run the program
./src/main
```
# MAKE Build
```bash
# clone the repo and cd to the root dir

# cd into the source dir
cd src

# run the make file
make

# run the program
./main.o
```
